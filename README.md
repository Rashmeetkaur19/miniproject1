# miniproject1

## Overview
This is a **Books Web Application** built with **Node.js** and **Express.js**.  
It displays a collection of books with their title, author, year, and cover image using JSON data.

---

## Features
- View books as **cards** with image and details.
- Responsive and clean design.
- Fetches data from a **JSON API** using JavaScript.
- Navigation between Home, About, Books API, and Contact pages.

---

## Technologies
- Node.js & Express.js
- HTML, CSS, JavaScript
- JSON for data
- Nodemon for development

---

## How to Run

1. npm install
2. npm start
3. http://localhost:3000/site/index.html
